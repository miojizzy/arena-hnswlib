- C++17
- CMake 3.15+
- GoogleTest (unit testing)
- Google Benchmark (performance benchmarking)

- Ubuntu 24.04.3 LTS

- Only depends on the standard library and GoogleTest/Benchmark
- Must support mainstream Linux platforms
